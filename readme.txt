Just a simple and light plugin made to redesign your dashboard.

== Description ==

Easy Dashboard is a simple plugin that helps you and your clients to start with WordPress. Once installed, you will find a new Dashboard, with some things wrapped up and some "metro-style" shortcuts to your menu.

== Installation ==

1. Download the plugin and install in WordPress
2. Activate it
